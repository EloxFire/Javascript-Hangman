Bienvenue !


Bienvenue sur ce jeu du pendu.
Les règles du jeu sont simples, deux modes de jeu vous sont proposés :
Joueur vs Ordinateur
Ordinateur vs Joueur

Dans le mode Joueur vs Ordinateur, un mot aléatoire est séléctionné par l'ordinateur et vous devez le deviner. ce mot peut contenir des lettres et/ou des chiffres

Dans le mode Ordinateur vs Joueur, c'est vous qui définissez le mot a trouver et l'ordinateur doit le deviner. Libre a vous de mettre des lettres et/ou des chiffres.

Sélectionnez un mode :
